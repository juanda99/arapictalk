# AraPicTalk

Sistema de comunicación aumentativa basado en ARASAAC.

## Despliegue

La aplicación se despliega automáticamente en GitHub Pages en cada commit a la rama `main`.

URL: [https://juanda99.github.io/arapictalk/](https://juanda99.github.io/arapictalk/)

## Características

- Basado en la suite `aratools`.
- Gestión de zoom dinámico.
- Ciclo de categorías individual por columna.
- Exportación a PDF e Imagen.
- Diseño totalmente responsivo y optimizado para tablets.
